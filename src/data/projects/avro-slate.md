Avro Slate is a Bengali text editor built on top of the [Slate.js](https://docs.slatejs.org/) framework, providing phonetic Avro-style typing for smooth Bangla writing. It was designed as a lightweight, focused notepad for Bengali users.

Key Features:

- **Avro Phonetic Typing**: Type Bangla easily using English letters with Avro-style phonetic mapping.

- **Lightweight Editor**: Clean and distraction-free interface for quick writing without heavy dependencies.

- **Persistent Drafts**: Keeps your text safe with local storage, so you won’t lose your notes accidentally.

- **Cross-Platform**: Runs entirely in the browser, no installation needed.

A minimal and simple editor for Bangla writing—just open, type, and save.
